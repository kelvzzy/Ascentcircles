## 💡 Pro Tips

### Before Saving:
1. **Compress images** at tinypng.com (keeps quality, reduces size)
2. **Rename exactly** as shown (case-sensitive!)
3. **Use .jpg format** (not .jpeg)

### File Sizes:
- Hero images: Under 500KB each
- Gallery images: Under 300KB each
- Total: Under 3MB for all images

### Quality Settings:
- Use 85% quality when compressing
- Keep resolution at 1920x1080 for hero
- 1200x800 is fine for gallery

## 🆘 Troubleshooting

**Image not showing?**
- Check filename matches exactly
- Check it's in `public/images/` folder
- Restart dev server
- Clear browser cache

**Image looks blurry?**
- Use higher resolution source
- Reduce compression (use 90% quality)

**Page loads slowly?**
- Compress images at tinypng.com
- Aim for under 500KB per image
