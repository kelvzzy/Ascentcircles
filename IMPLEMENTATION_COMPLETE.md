#### 2. Configure Services (15 minutes)

**Create `.env.local` file:**
```bash
# Google Analytics (get from analytics.google.com)
NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX

# WhatsApp (your number with country code, no + or spaces)
NEXT_PUBLIC_WHATSAPP_NUMBER=2348012345678
```

**Update WhatsApp number:**
- Open `src/components/WhatsAppButton.tsx`
- Line 18: Change `2348012345678` to your actual number

#### 3. Deploy (10 minutes)
```bash
# Build for production
npm run build

# Deploy to Vercel (recommended)
vercel deploy --prod

# Or deploy to Netlify
netlify deploy --prod
```

## 🔧 Troubleshooting

### Analytics Not Working?
- Check `.env.local` has correct GA ID
- Verify it starts with `G-`
- Wait 24 hours for data to appear
- Check in Incognito mode (extensions can block)

### WhatsApp Button Not Working?
- Verify number format: `2348012345678`
- No + symbol, no spaces
- Include country code (234 for Nigeria)
- Test on mobile device

### Forms Not Submitting?
- Check browser console for errors
- Verify all required fields filled
- Email must be valid format
- Try different browser

### Images Not Loading?
- Check file names match exactly
- Files must be in `public/images/`
- Use .jpg format (not .jpeg or .png)
- Refresh browser cache (Ctrl+Shift+R)
