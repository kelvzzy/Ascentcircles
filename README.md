# The Ascent Circle Landing Page

A visually stunning, emotionally compelling, high-conversion landing page for The Ascent Circle - a global movement rooted in nature, built on presence, rising together.

## 🌟 Features

- **Cinematic Hero Section** with video background and parallax effects
- **Authentic Storytelling** that connects emotionally with visitors
- **Five Core Pillars** showcasing what makes the movement different
- **Experience Gallery** with immersive content preview
- **Global Expansion Map** showing local circles and growth
- **Partnership Opportunities** for brands and sponsors
- **Volunteer Recruitment** for Path Keepers
- **Email Capture** with elegant form design
- **Comprehensive Footer** with values and contact information

## 🎨 Design System

### Colors
- **Forest Green**: #2D5016 (Primary)

- **Warm Gold**: #DAA520 (Accent)

- **Headlines**: Crimson Text (Serif)

### Animations
- Framer Motion for smooth transitions
- Scroll-based animations
- Micro-interactions

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd ascent-circle-landing
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
