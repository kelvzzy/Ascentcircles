# Logo Setup Instructions

## Save the Logo

Please save the Ascent Circle logo image to:
```
ascent-circle-landing/public/logo.png
```

The logo should be the circular badge design with:
- Two hikers helping each other up a mountain
- Sunset/sunrise background with golden rays
- "THE ASCENT CIRCLE" text
- "We Rise Together" tagline
- Mountain and tree silhouettes
- Moon, stars, and sun elements

## Color Palette Extracted from Logo

The website now uses these exact colors from the logo:

### Primary Colors
- **Forest Green/Teal**: `#1a3d3d` (dark backgrounds, header, footer)
- **Forest Light**: `#2d5555` (hover states)

### Sunset/Sunrise Colors
- **Sunset Orange**: `#d97634` (primary CTAs, accents)
- **Sunset Light**: `#e88a3f` (hover states)
- **Sunset Lighter**: `#f5a855` (highlights)

### Golden/Yellow Colors
- **Gold**: `#e8b968` (text highlights, borders)
- **Gold Light**: `#f4d89f` (backgrounds, hover)
- **Gold Lighter**: `#fef5e7` (light backgrounds)

### Cream/Beige Colors
- **Cream**: `#f5e6c8` (text on dark backgrounds)
- **Cream Light**: `#fef5e7` (section backgrounds)

### Sage/Mountain Green
- **Sage**: `#7a9b8e` (accents, decorative elements)
- **Sage Light**: `#8fae9d` (hover states)

## Where the Logo Appears

1. **Header** - Top left with "The Ascent Circle" text and "We Rise Together" tagline
2. **Footer** - Can be added if desired
3. **Favicon** - Should be created from the logo

## Next Steps

1. Save the logo to `public/logo.png`
